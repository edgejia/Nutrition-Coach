## Trust Boundaries

| Boundary | Description |
|---|---|
| Test contracts -> production implementation | Red/source assertions must pin security-sensitive UI/transport behavior without hand-waving implementation. |
| Browser -> grouped PATCH transport | Grouped request body construction is untrusted until the server validates it. |
